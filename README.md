# Event Curator — Auckland Events

A single-page web app for browsing upcoming events in Auckland. Open `Auckland Events.html` directly in any browser — no server needed.

---

## Run

Just open the file:

```bash
open "Auckland Events.html"
```

Or double-click it in Finder.

---

## Notes

- No dependencies, no build step
- All styles and logic are self-contained in the HTML file
